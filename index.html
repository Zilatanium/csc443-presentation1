2.3: Development Environment in Relation to the Public Internet?

    Not all Environments are the Same

    People all around the world connected to the internet may be using a number of different combinations of IDE, OS, Browser, etc... This makes it important to have some sort of agreed on use of transferring data from on computer to the next. An example of this would be the Open System Interconnection Model:

  • Physical Layer - Raw bit stream
  • Data Link Layer - Format of data
  • Network Layer - Physical path
  • Transport Layer - Transmits data
  • Session Layer - Controls ports and session
  • Presentation Layer - Data is in usable format
  • Application - Human-computer interaction (Environment)
